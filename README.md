## 왜 VZZK인가요?

### 🎯 흩어진 기능, 하나로 모았습니다

투표는 A 사이트, 음악 신청은 B 사이트, TTS는 C 프로그램...

**VZZK 하나면 전부 됩니다.**

- 탭 10개 띄워놓고 왔다 갔다 할 필요 없음
- 사이트마다 다른 설정 방식 익힐 필요 없음
- 한 화면에서 모든 기능을 켜고 끄고 설정

### 🔒 내 데이터는 내 PC에만

VZZK은 모든 데이터를 로컬에 저장합니다.

- 후원 금액, 시청자 정보 등 민감한 데이터가 외부 서버로 전송되지 않음
- 계정 정보는 내 컴퓨터에만 암호화되어 저장
- 서비스 종료 걱정 없이 내 설정 그대로 유지

### ⚡ 5분이면 방송 세팅 끝

1. 다운로드
2. 네이버 로그인
3. 원하는 기능 켜기
4. 끝!
